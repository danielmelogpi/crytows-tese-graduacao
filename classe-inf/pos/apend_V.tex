\chapter{Ambiente de desenvolvimento utilizado no experimento}
\label{apend:5}

Abaixo são dispostos itens que descrevem o ambiente de desenvolvimento utilizado e as ferramentas que fizeram parte do processo de desenvolvimento deste trabalho e do experimento associado.

\begin{itemize}
\item Java 8 (OpenJDK 8) \verb|1.8.0_111-b14| : Linguagem de desenvolvimento principal
\item Git para Linux 2.7.4 : Ferramenta de controle de versão do código-fonte
\item Github : Plataforma em nuvem para compartilhamento de projetos Git
\item Linux Ubuntu Kernel 4.4.0-47-generic 64 bits : Sistema Operacional utilizado para desenvolvimento
\item Android Studio 2.2.2 : Ambiente de desenvolvimento para projetos Android
\item Visual Studio Code 1.7.2 -  Editor simples de texto para atividades auxiliares
\end{itemize}
