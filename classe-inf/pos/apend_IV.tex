\chapter{Detalhes de itens de configuração do aplicativo}
\label{apend:4}

Abaixo estão expostos detalhes de versão e descrição de linguagens, bibliotecas e aplicações externas utilizadas no desenvolvimento do aplicativo móvel que implementa a proposta deste trabalho.

\begin{itemize}
\item Java 8 (OpenJDK 8)  \verb|1.8.0_111| : Linguagem de programação para plataforma Android
\item Android SDK Level 22: Kit de desenvolvimento para a API Android 5.1 Lollipop
\item Gradle  2.14.1: Gestão das dependências
\item OpenPGP API 11 : Cliente de APIs PGP oferecidas por aplicativos de dispositivos móveis
\item OpenKeyChain 4.1 : Aplicativo de código aberto que implementa operações de chaveiro PGP
\item Apache Commons IO 2.4 : Biblioteca Java para abstração de operações com Streams, dentre outras funções
\item Java NanoHTTPD 2.3.1 : Biblioteca que permite a implementação de servidor HTTP, abstraindo o recebimento e a resposta das requisições dentro do aplicativo.
\item Jackson JR  2.8.5 :Implementação menor da biblioteca Jackson Databind, para serialização e deserialização de JSON
\end{itemize}
